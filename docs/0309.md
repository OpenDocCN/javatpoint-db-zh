# MySQL 第一个函数

> 原文：<https://www.javatpoint.com/mysql-first>

MySQL first 函数用于返回所选列的第一个值。这里，我们使用 limit 子句选择第一条记录或更多记录。

**语法:**

```
SELECT column_name
FROM table_name
LIMIT 1;

```
