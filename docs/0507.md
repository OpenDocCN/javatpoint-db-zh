# 蒙古数据库运营商

> 原文:[https://www.javatpoint.com/mongodb-lte-operator](https://www.javatpoint.com/mongodb-lte-operator)

## MongoDB 中的$lte 运营商是什么？

MongoDB 提供了多种比较查询操作符。$lte(小于等于)运营商就是其中之一。$lte 运算符用于选择字段值小于或等于(即< =)给定值的文档。您可以根据需要在更新、查找等方法中使用该运算符。

### $lte 运营商的语法:

```

{ field : { $lte : value } }

```
