# 甲骨文或

> 原文:[https://www.javatpoint.com/oracle-or](https://www.javatpoint.com/oracle-or)

在 Oracle 中，OR 运算符用于检查多个条件，并在任何一个给定条件为真时返回结果。

## 句法

```

WHERE condition1
OR condition2
...
OR condition_n;

```

## 因素

**条件 1，条件 2，条件 n :** 如果任一条件为真，则返回记录。
