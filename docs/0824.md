# 比较运算符

> 原文:[https://www.javatpoint.com/oracle-comparison-operators](https://www.javatpoint.com/oracle-comparison-operators)

在 Oracle 中，比较运算符与 where 子句一起使用。以下是可以使用的运算符:-

| 比较运算符 | 描述 |
| = | 平等的 |
| <> | 不平等 |
| ！= | 不平等 |
| > | 大于 |
| >= | 大于或等于 |
| < | 不到 |
| <= | 小于或等于 |
