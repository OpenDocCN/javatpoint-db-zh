# 马里亚数据库最大值()函数

> 原文:[https://www.javatpoint.com/mariadb-max-function](https://www.javatpoint.com/mariadb-max-function)

MariaDB MAX()函数用于检索表达式的最大值。

**语法:**

```
SELECT MAX(aggregate_expression)
FROM tables
[WHERE conditions]; 

```
