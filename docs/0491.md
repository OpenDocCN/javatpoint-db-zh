## MongoDB 中的$divide 运算符是什么？

MongoDB 提供了多种算术表达式运算符。$divide 运算符就是其中之一。此运算符用于聚合管道阶段。它也用于将一个数除以另一个数并返回结果。

### $divide 运算符的语法:

```

{ $divide : [ < expression 1 >, < expression 2 > ] }

```

要使用此运算符，您必须在数组中传递参数。在这个运算符中，第一个参数是被除数，第二个参数是除数。

**示例 1:使用$divide 运算符**

在本例中，我们将使用$divide 运算符来查找手机的价格。

```

db.products.aggregate(
 [
   {$match: { name : "Mobile"}},
      {
          $project: 
              {
                name : 1,
                totalPrice : 1,
                totalQuantity : 1,
                mobilePrice : {$divide: ["$totalPrice", "$totalQuantity"]}
               }
      }
 ]
)

```

**输出:**

```
{
        "_id" : 1,
        "name" : "Mobile",
        "totalPrice" : 100000,
        "totalQuantity" : 50,
        "mobilePrice" : 2000
}
{
        "_id" : 5,
        "name" : "Mobile",
        "totalPrice" : 20000,
        "totalQuantity" : 4,
        "mobilePrice" : 5000
}
{
        "_id" : 6,
        "name" : "Mobile",
        "totalPrice" : 25000,
        "totalQuantity" : 2,
        "mobilePrice" : 12500
}

```

**例 2:添加自己的号码**

如果用户想用某个数字来划分一个字段，用户可以这样做。在本例中，我们将把“总价”字段除以 2。

```

db.products.aggregate(
 [
   {$match: { name : "Memory Card"}},
      {
          $project: 
              {
                name : 1,
                totalPrice : 1,
                halfPrice : { $divide : [ "$totalPrice", 2 ]}
               }
      }
 ]
)

```

**输出:**

```
{
        "_id" : 4,
        "name" : "Memory Card",
        "totalPrice" : 2500,
        "halfPrice" : 1250
}
{
        "_id" : 7,
        "name" : "Memory Card",
        "totalPrice" : 1000,
        "halfPrice" : 500
}
{
        "_id" : 10,
        "name" : "Memory Carde",
        "totalPrice" : 4000,
        "halfPrice" : 2000
}

```

**例 3:数据类型错误**

$divide 运算符只支持整数数据类型。如果该值不是整数，则会产生错误。

```

db.products.aggregate(
 [
   {$match: {name : "Pen drive"}},
      {
          $project: 
              {
                item_name : 1,
                total_Price : 1,
```

**示例 4:在嵌入文档中使用$divide 运算符**

在这个例子中，我们将笔记本电脑的总价格除以 2。

```

db.products.aggregate(
 [
   {$match: { name : "Laptop"}},
      {
          $project: 
              {
                name : 1,
                halfPrice : {$divide: ["$billDetail.totalPrice", 2]}
               }
      }
 ]
)

```

**输出:**

```
{
        "_id" : 9,
        "name" : "Laptop",
        "halfPrice" : 22500
}

```
