# 甲骨文不为空

> 原文:[https://www.javatpoint.com/oracle-is-not-null](https://www.javatpoint.com/oracle-is-not-null)

在 oracle 中，IS NOT NULL 用于检查非空值。它与 select、insert、update 和 delete 语句一起使用。

## 句法

```

expression IS NOT NULL

```

## 因素

**表达式:**列名或任何要检查的值都不是空值

### 注意:

1.  在“不为空”中，条件返回真值。
2.  在空值中，条件返回假。
