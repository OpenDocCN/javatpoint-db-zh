# 在 Windows 上安装 Neo4j

> 原文:[https://www.javatpoint.com/install-neo4j](https://www.javatpoint.com/install-neo4j)

## 下载 Neo4j

先从其官网下载 Neo4j:[https://neo4j.com/download/](https://neo4j.com/download/)

您可以选择免费的企业试用版或免费的社区版。在这里，我们使用社区版。
