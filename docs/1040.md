# MariaDB 订单子句

> 原文：<https://www.javatpoint.com/mariadb-order-by>

在 MariaDB 中，ORDER BY 子句用于按升序或降序对结果集中的记录进行排序。

**语法:**

```

SELECT expressions
FROM tables
[WHERE conditions]
ORDER BY expression [ ASC | DESC ]; 

```

#### 注意:您可以不使用 ASC/DESC 属性对结果进行排序。默认情况下，结果将按升序排序。
