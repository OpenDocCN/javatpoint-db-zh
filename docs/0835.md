# ORACLE where 子句

> 原文:[https://www.javatpoint.com/oracle-where](https://www.javatpoint.com/oracle-where)

在 Oracle 中，where 子句用于筛选结果。Where 子句可用于 INSERT、UPDATE、DELETE 和 SELECT 语句。

## 句法

```

WHERE conditions;

```

## 因素

**获取指定记录的条件:**。
