# SQLite 联接

> 原文:[https://www.javatpoint.com/sqlite-joins](https://www.javatpoint.com/sqlite-joins)

在 SQLite 中，JOIN 子句用于组合数据库中两个或多个表的记录。它通过使用两个表的公共值来联合两个表中的字段。

SQLite 中主要有三种类型的连接:

*   SQLite INNER JOIN
*   SQLite OUTER JOIN
*   SQLite CROSS JOIN
