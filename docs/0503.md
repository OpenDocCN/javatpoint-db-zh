# MongoDB $gt 运算符(大于)

> 原文：<https://www.javatpoint.com/mongodb-gt-operator>

## MongoDB 中的$gt 运算符是什么？

MongoDB 提供了多种比较查询操作符。$gt(大于)运算符就是其中之一。$gt 运算符用于选择字段值大于给定值的文档。您可以根据需要在(更新、查找等)方法中使用该运算符。

### $gt 运算符的语法:

```

{ field : { $gt : value } }

```
